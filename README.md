# OpenCode 全局配置

这是我的 [OpenCode](https://opencode.ai) 全局配置目录 (`~/.config/opencode`)。

## 使用

1. 将此目录放在 `~/.config/opencode`
2. 运行 `npm install` 安装插件依赖
3. 编辑 `opencode.jsonc` 配置模型和提供商
