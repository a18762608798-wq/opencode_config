# pandas groupby.agg 分组聚合与大数据分块处理

**日期**: 2026-07-19 20:42
**分类**: 编程/Python
**标签**: #pandas #groupby #agg #大数据 #chunksize

## 背景

用户需要处理 50 万行的 CSV 数据，按 `category` 列分组后计算 `amount` 列的均值和标准差，同时关心大文件的内存效率。

## 内容

### 基本用法

```python
df.groupby('category')['amount'].agg(['mean', 'std'])
```

`agg` 一次调用即可完成多个聚合函数，输出多级列索引的 DataFrame。

### 大文件分块读取

当内存不足以一次性加载整个文件时，使用 `chunksize` 参数分块读取：

```python
chunks = []
for chunk in pd.read_csv('data.csv', chunksize=50000):
    result = chunk.groupby('category')['amount'].agg(['mean', 'std'])
    chunks.append(result)

# 合并各分块结果
final = pd.concat(chunks).groupby(level=0).mean()
```

### 内存效率说明

`agg` 在一次分组操作中完成多个聚合，比先 `groupby` 再分别调用 `.mean()` 和 `.std()` 更高效，因为前者只需遍历数据一次。

## 要点

- `agg` 支持传入聚合函数名列表，一次完成多种聚合
- 大 CSV 文件用 `chunksize` 参数分块读取，迭代器返回每个 block
- `agg` 方法比分开调用多个聚合函数内存效率更高
- 分块后需用 `pd.concat` + 二次聚合合并结果
