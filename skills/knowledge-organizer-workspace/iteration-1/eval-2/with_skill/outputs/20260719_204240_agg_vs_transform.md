# pandas agg 与 transform 的区别

**日期**: 2026-07-19 20:42
**分类**: 编程/Python
**标签**: #pandas #groupby #agg #transform

## 背景

用户在学习 pandas 分组聚合时，对 `agg` 和 `transform` 的差异产生疑问。

## 内容

### 核心区别

| 方法 | 返回形状 | 用途 |
|------|---------|------|
| `agg` (aggregate) | 行数 = 分组数 | 将每组数据压缩为单个标量值 |
| `transform` | 行数 = 原始行数 | 将聚合值广播回每一行，保持原始形状 |

### agg 示例

```python
df.groupby('category')['amount'].agg(['mean', 'std'])
# 结果行数 = category 的不同取值个数
```

### transform 示例

```python
df['zscore'] = df.groupby('category')['amount'].transform(
    lambda x: (x - x.mean()) / x.std()
)
```

`transform` 计算每组内的 z-score，但结果长度与原始 DataFrame 一致，可以直接作为新列添加回原 DataFrame。

### 使用场景

- **agg**: 需要汇总统计，减少数据量（如生成报表）
- **transform**: 需要保留原始行，但使用组级统计量（如标准化、缺失值用组均值填充、计算组内排名百分比等）

## 要点

- `agg` 输出行数等于分组数，`transform` 输出行数等于原始行数
- `transform` 结果可直接加回 DataFrame 作为新列
- 需要保留原始数据的前提下做组级计算时用 `transform`
- 做汇总报表时用 `agg`
