# 量子纠缠的本质与在量子算法中的角色

**Date**: 2026-07-19 12:00
**Category**: Theory/Quantum Computing
**Tags**: #entanglement #bell-state #quantum-correlation

## Background
在讨论量子计算加速原理时进一步深入探讨了量子纠缠的作用与本质。

## Content
量子纠缠是量子系统之间一种独特的非经典关联，其核心特征在于：复合系统的量子态不能分解为各个子系统态的简单张量积。

典型例子是 Bell 态：
```
|Φ⁺⟩ = (|00⟩ + |11⟩) / √2
```
对该纠缠态中第一个量子比特进行测量，其结果会瞬间决定第二个量子比特的状态。这种关联无法用经典概率解释。

在 Grover 算法中，纠缠并非必须——该算法在单量子比特系统上即可运行。然而，纠缠是许多其他量子算法和协议的关键资源，包括 Shor 算法（大数分解）和量子隐形传态。更宏观地看，量子纠缠是量子关联的最强形式，可以超越经典关联的 Bell 不等式上限（即 CHSH 不等式上限 2，量子力学可达 2√2 ≈ 2.83）。

## Key Points
- 纠缠态不可分解为子系统态的张量积，这是其数学定义
- Bell 态是最简单的最大纠缠两体态，CHSH 不等式用于检验纠缠
- Grover 无需纠缠；Shor、量子隐形传态等依赖纠缠
- 纠缠超越经典关联上限，体现量子非定域性
- 纠缠 + 叠加 + 干涉构成量子计算的三大核心资源
