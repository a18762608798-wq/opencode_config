# 量子比特与经典比特的本质区别

**Date**: 2026-07-19 12:00
**Category**: Theory/Quantum Computing
**Tags**: #qubit #superposition #quantum-mechanics

## Background
量子计算入门者询问量子比特和经典比特之间的核心差异。

## Content
经典比特只能处于 0 或 1 两种确定状态之一，而量子比特可以处于 0 和 1 的叠加态，这是两者最本质的区别。

数学上，一个量子比特的状态表示为：
```
|ψ⟩ = α|0⟩ + β|1⟩
```
其中 α 和 β 是复数振幅，满足归一化条件 |α|² + |β|² = 1。

关键理解：叠加态并不意味着量子比特"同时处于 0 和 1"，而是指测量的概率性——对量子比特进行测量时，以 |α|² 的概率得到结果 0，以 |β|² 的概率得到结果 1。测量会导致量子态坍缩到对应的基态。

## Key Points
- 经典比特是确定的二值系统（0 或 1），量子比特是连续概率幅系统
- Bloch 球面表示：经典比特只在两极，量子比特可取球面上任意点
- 叠加态的概率性本质：|α|² + |β|² = 1 定义了测量结果的概率分布
- 测量导致波函数坍缩，这是量子力学与经典物理的根本差异之一
