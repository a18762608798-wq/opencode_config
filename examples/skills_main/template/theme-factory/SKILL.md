# 主题工厂技能

此技能提供一组精选的专业字体和颜色主题，每个主题都配有精心挑选的调色板和字体搭配。选定主题后，可将其应用于任何制品。

## 用途

为演示幻灯片等制品应用一致的专业风格。每个主题包含：
- 带十六进制代码的统一调色板
- 标题和正文的互补字体搭配
- 适合不同上下文和受众的独特视觉形象

## 使用说明

为幻灯片或其他制品应用风格：

1. **展示主题展示**：显示 `theme-showcase.pdf` 文件，让用户直观查看所有可用主题。不要对其进行任何修改；仅展示文件以供查看。
2. **询问选择**：询问选择哪个主题应用于制品
3. **等待选择**：获得关于所选主题的明确确认
4. **应用主题**：选定主题后，将所选主题的颜色和字体应用到制品上

## 可用主题

以下 10 个主题可用，每个主题都在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** - 专业沉稳的海洋主题
2. **Sunset Boulevard** - 温暖活力的日落色彩
3. **Forest Canopy** - 自然沉稳的大地色调
4. **Modern Minimalist** - 干净现代的灰度风格
5. **Golden Hour** - 丰富温暖的秋季调色板
6. **Arctic Frost** - 冷冽清爽的冬季灵感主题
7. **Desert Rose** - 柔和精致的灰粉色调
8. **Tech Innovation** - 大胆现代的科技美学
9. **Botanical Garden** - 清新有机的花园色彩
10. **Midnight Galaxy** - 戏剧性宇宙深色调

## 主题详情

每个主题在 `themes/` 目录中定义，包含完整规格：
- 带十六进制代码的统一调色板
- 标题和正文的互补字体搭配
- 适合不同上下文和受众的独特视觉形象

## 应用流程

选定首选主题后：
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个制品中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉一致性

## 创建自定义主题
当没有现成主题适合制品时，创建自定义主题。根据提供的输入，生成一个与上述类似的新主题。为主题取一个类似风格的名字，描述其字体/颜色组合所代表的风格。使用提供的基本描述来选择适当的颜色/字体。生成主题后，展示以供审查和验证。之后，按上述方式应用该主题。
